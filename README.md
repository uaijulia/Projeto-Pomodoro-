#### 🍅 Pomodoro Timer

Pomodoro Timer minimalista e responsivo feito em **HTML, CSS e JavaScript** para foco, produtividade e estudo eficiente.

---

#### 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Lógica de programação com JavaScript puro
- Manipulação do DOM
- Estruturação de interface responsiva
- Boas práticas de organização de código front-end

Além disso, o projeto simula um aplicativo de produtividade baseado na técnica Pomodoro.

---

#### ⚙️ Funcionalidades

- Timer de foco Pomodoro
- Iniciar, pausar e resetar o contador
- Layout responsivo (desktop e mobile)
- Interface minimalista e focada em usabilidade
- Persistência de dados com **localStorage** (o progresso não se perde ao atualizar ou fechar a página)

---

#### 🧠 Armazenamento de dados

Este projeto utiliza o **localStorage do navegador** para salvar informações localmente.

Isso significa que:
- Os dados permanecem salvos mesmo após fechar a página
- Não há necessidade de banco de dados externo
- Tudo é armazenado diretamente no navegador do usuário

---

- HTML5

- JavaScript
